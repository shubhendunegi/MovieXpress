A PERN movie database app to fetch from backend local api made with PostgreSQL, Express, React and Node..

Librabies/ Frameworks:
React
PG- pool (postgres)
Express
Bootstrap

List of APIs created:

Backend-
-GET all movies
-GET one movie
-Create a movie (POST)
-Update a movie (PUT)
-DELETE a movie

Frontend-
-MovieFinder (api to connect to backend)
